Backend protocol

Rest AMQ
Select the backend handler for PUT, DELETE, POST operations
- Rest: request will be forward to rest interface
- AMQ: requests will be forward to RabbitMQ interface
** AMQ requests will return success if producer success forwards MSG to consumers

Functionality

* Interface for CRUD operations on Booking items * Table supports: server side Paging, Sorter and Filter * From Table users can access Edit page * Edit page actions: Edit, Delete, Clone * State management: Akita is used as state management
